Notes:

Alignment on the Main Axis
  • justify-content - the whole content of a container - flex-start, flex-end, center, space-between, space-around, baseline etc
  • justify-items - NOT available for Flexbox
  • justify-self - NOT available for Flexbox
Alignment on the Cross Axis
  • Requires a height (for row direction) to be set to be effective
  • Properties starts with align...:
  • align-content - the whole content of a container - start, end, center, space-between, space-around, baseline etc
  • align-self - on flex (child) items - flex-start, flex-end, center, stretch - default stretch (except images)
  • align-items - on container - like align-self, for all children
