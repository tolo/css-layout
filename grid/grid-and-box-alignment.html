Notes:

Alignment on the Block Axis
  • For grid - this means vertical (top-down*)
  • Requires a height to be set to be effective
  • Properties starts with align...:
  • align-content - the whole content of a container - start, end, center, space-between, space-around, baseline etc
  • align-self - on child items - start, end, center, stretch - default stretch (except images)
  • align-items - on container - like align-self, for all children
Alignment on the Inline Axis
  • For grid - this means horizontal (left-right*)
  • Properties starts with justify...:
  • justify-content - the whole content of a container - start, end, center, space-between, space-around, baseline etc
  • justify-items - on child items - start, end, center, stretch - default stretch (except images)
  • justify-self - on container - like align-self, for all children
  • place-self - shorthand for both justify-self and align-self, not widely supported yet
Generic and easy way to center an item:
display: grid;
justify-content: center;
align-items: center;
